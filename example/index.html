"jQuery Lettering Animate" is an extension of the "lettering plugin", it iterates through each letter separated by the lettering and animates it.

Here is the sintax to use it:

$("#element").lettering().animateLetters({randomOrder:true,time:1000},{left:200,opacity:0},onCompleteHandler);

First you apply the lettering effect to the dom element, then, you animate the letters passing as parameters the plugin options "randomOrder" and "time",then, you set the intro or the outro animation properties properties of each letter, and the "onComplete" callback.

Requirements:

  • jQuery Library
  • Lettering Plugin
  • Lettering Animate Plugin

Examples:

This is the example animating the opacity property

$("#title01").lettering().animateLetters({randomOrder:false,time:500},{opacity:0}); $("#title01").lettering().animateLetters({randomOrder:false,time:500},null,{opacity:0});

This is the example animating the opacity property in random order

$("#title01r").lettering().animateLetters({randomOrder:true,time:1000},{opacity:0}); $("#title01r").lettering().animateLetters({randomOrder:true,time:1000},null,{opacity:0});

This is the example animating the top and opacity property

$("#title02").lettering().animateLetters({randomOrder:false,time:500},{top:20,opacity:0}); $("#title02").lettering().animateLetters({randomOrder:false,time:500},null,{top:20,opacity:0});

This is the example animating the top and opacity property in random order

$("#title02r").lettering().animateLetters({randomOrder:true,time:1000},{top:20,opacity:0}); $("#title02r").lettering().animateLetters({randomOrder:true,time:1000},null,{top:20,opacity:0});

This is the example animating the left and opacity property

$("#title03").lettering().animateLetters({randomOrder:false,time:500},{left:-200,opacity:0}); $("#title03").lettering().animateLetters({randomOrder:false,time:500},null,{left:-200,opacity:0});

This is the example animating the left and opacity property in random order

$("#title03r").lettering().animateLetters({randomOrder:true,time:1000},{left:-200,opacity:0}); $("#title03r").lettering().animateLetters({randomOrder:true,time:1000},null,{left:-200,opacity:0});

This is the example animating the top, left and opacity property

$("#title04").lettering().animateLetters({randomOrder:false,time:500},{top:200,left:200,opacity:0}); $("#title04").lettering().animateLetters({randomOrder:false,time:500},null,{top:200,left:-200,opacity:0});

This is the example animating the top, left and opacity property in random order

$("#title04r").lettering().animateLetters({randomOrder:true,time:1000},{top:200,left:200,opacity:0}); $("#title04r").lettering().animateLetters({randomOrder:true,time:1000},null,{top:200,left:-200,opacity:0});
